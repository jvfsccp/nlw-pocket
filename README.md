# Criador de Metas
Feito no evento NlW-Pocket JavaScript feito pela RocketSeat e sendo instruido pelo Mayk Brito.
<br>
<br>
Seguintes funcionalidades do sistema:
- Criar metas
- Listas metas
    - Metas realizadas
     - Metas abertas
-  Marcar / Desmarcar metas realizadas
- Remover metas
- Sistema de mensagens 
- Persistir dados
